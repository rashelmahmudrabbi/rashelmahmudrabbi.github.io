HYBSWINEFF – Hybrid CNN Transformer Fusion for Blood Cell Cancer Classification

Developed a novel hybrid deep learning framework combining CNN feature extraction with Transformer-based contextual learning for binary and multi-stage blood cell cancer classification. The proposed architecture enhances feature representation, improves classification robustness, and supports clinically interpretable medical image analysis. The extended version of this thesis is currently being prepared for international journal publication in the field of AI-driven healthcare and medical imaging.

HYBSWINEFF: Hybrid CNN-Transformer Fusion for Binary and Multi-Stage Blood Cell Cancer Classification

Created a hybrid CNN-Transformer model that performs binary and multi-stage leukemia classification of peripheral blood smear images. Incorporates Swin Transformer with CNN feature fusion and explainable AI methods (Grad-CAM, LIME) to make predictions interpretable for clinical use. Targets automated leukemia screening from microscopic blood cell images.

PyTorch Swin Transformer CNN Grad-CAM LIME OpenCV

MRCL-ELM – Multi-Class Satellite Image Classifier
Novel hybrid deep learning approach for satellite image classification achieving 98.33% accuracy on EuroSAT and 98.10% on UC Merced. Integrates LIME and SHAP for explainability; deployed as a real-time web application. Submitted to a Q1 journal.
